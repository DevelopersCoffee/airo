# Airo Roadmap

This roadmap describes planned Airo TV milestones. Scope and timing can change as issues are validated.

## Airo TV

### v0.0.x

- Local and remote M3U playlist loading.
- Android TV launcher support.
- Channel search.
- Basic playback.
- Cast controls and diagnostics.
- Release checksums and professional release notes.

### v0.1

- Favorites.
- Playlist management improvements.
- Screenshot and demo-video release assets.
- Expanded device qualification reports.

### v0.2

- EPG support.
- Program metadata display.
- Improved error recovery for unsupported streams.

### v0.3

- AI-assisted search experiments.
- Better content grouping and recommendations from local playlist metadata.

### v0.4

- Multiple playlist management.
- Import/export improvements.

### v0.5

- Store-ready onboarding and settings hardening.
- Broader TV device compatibility validation.

### v1.0

- Stable Android TV release with documented support policy.
- Reproducible release process with signed tags and published checksums.

## Not Planned for Current 0.x Releases

- Bundled IPTV channels.
- Recording or DVR functionality.
- Hosting or proxying IPTV content.
