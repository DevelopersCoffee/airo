# Generated XMLTV Benchmark Fixture

Fixture id: `generated-xmltv-50mb-v1`

This directory contains a deterministic synthetic XMLTV fixture for large-guide
EPG/parser benchmark work. It is generated data only; it does not contain
provider guide URLs, customer data, credentials, or local machine paths.

## Files

| File | Purpose |
| --- | --- |
| `generated-50mb.xml` | 50 MiB synthetic XMLTV fixture. |
| `generated-50mb.manifest.json` | Machine-readable fixture metadata. |

| Channels | `512` |
| Programmes | `83209` |
| Programme duration minutes | `30` |
| Start UTC | `2026-01-01T00:00:00.000Z` |

## Regeneration

From the workspace root:

```bash
dart run melos run bench:xmltv-fixture
```

The generator lives in `packages/platform_benchmarks` so future Rust/XMLTV
engine work can reuse the same fixture contract without putting benchmark data
generation in Airo TV application code.
