### Adaptive Navigation

| Platform | Navigation Type | Implementation |
|----------|-----------------|----------------|
| Mobile | Bottom Navigation Bar | 4 tabs: Dashboard, Groups, Mind, Settings |
| Tablet | Navigation Rail | Collapsible, icons + labels |
| Desktop | Expanded Navigation Rail | Always visible, full labels |
| Web | Navigation Rail + Breadcrumbs | Context-aware navigation |

### Adaptive Components

```dart
// Example: Adaptive dialog for expense confirmation
class AdaptiveExpenseConfirmation extends StatelessWidget {
  final Expense expense;

  @override
  Widget build(BuildContext context) {
    return AdaptiveDialog(
      // Mobile: BottomSheet
      // Tablet/Desktop: Dialog
      mobileBuilder: (context) => ExpenseConfirmationSheet(expense),
      desktopBuilder: (context) => ExpenseConfirmationDialog(expense),
    );
  }
}

// Example: Responsive grid for budget categories
class BudgetCategoryGrid extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return ResponsiveGrid(
      columns: ResponsiveValue<int>(
        context,
        defaultValue: 2,
        conditionalValues: [
          Condition.largerThan(name: MOBILE, value: 3),
          Condition.largerThan(name: TABLET, value: 4),
        ],
      ),
      children: budgetCategories.map((c) => BudgetCategoryCard(c)).toList(),
    );
  }
}
```

## 11. Component Library

### Shared Components

| Component | Description | Usage |
|-----------|-------------|-------|
| `AmountDisplay` | Formatted currency display with sign coloring | Dashboard, Cards, Lists |
| `CategoryIcon` | Emoji + color background for categories | Expenses, Budgets |
| `ProgressBar` | Animated linear progress with threshold colors | Budgets, Goals |
| `MemberAvatar` | Circular avatar with initials fallback | Groups, Splits |

| `ActionCard` | Tappable card with icon, title, subtitle | Lists |

| `LoadingState` | Skeleton loaders matching layout | All screens |

## 12. Accessibility Requirements

### WCAG 2.1 AA Compliance

| Requirement | Implementation |
|-------------|----------------|
| Color contrast | Minimum 4.5:1 for text, 3:1 for large text |
| Touch targets | Minimum 48x48dp for all interactive elements |
| Screen reader | Semantic labels for all components |
| Focus indicators | Visible focus rings for keyboard navigation |
| Motion | Respect `prefers-reduced-motion` setting |

### Screen Reader Annotations
