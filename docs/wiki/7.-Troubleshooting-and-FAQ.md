# Troubleshooting and FAQ

## App Does Not Install on Android

- Confirm the APK downloaded completely.
- Allow installs from the browser or file manager used to open the APK.
- Try ADB installation with `adb install -r app-release.apk`.

## Login Redirects Me Back to Sign In

Airo protects most routes behind authentication. Sign in or create an account
before opening Dashboard, Assistant, Music, TV, Games, or Tasks.

## On-Device AI Is Not Available

Not every device supports Gemini Nano. If unsupported, Airo can show cloud AI
mode. On-device availability depends on device model, OS support, and runtime
configuration.

## Assistant Command Did Not Work

Try one of the known prompt cards first. If a message is not recognized as a
supported local command, Airo treats it as a general AI chat prompt.

## Quest Upload Failed

- Check that the file type is supported: PDF, JPG, JPEG, PNG, TXT, DOC, or DOCX.
- Try a smaller file.
- Reopen the Tasks tab and create a new quest.

## TV or Music Playback Fails

- Check network access for stream URLs.
- Try a different channel or track.
- Restart the app if the player state appears stuck.

## FAQ

**Q: Is Airo only an AI app?**
A: No. Airo combines AI with finance, file tasks, media, games, and app actions.

**Q: Does Airo work offline?**
A: Some workflows are designed for local or offline execution, especially
on-device AI where supported. Media streaming, downloads, cloud fallback, and
release installation require network access.

**Q: What games are available?**
A: Chess and Blackjack are documented as available. Additional games are planned
or documented as coming soon.

**Q: Where should I report issues?**
A: Use GitHub Issues from the Feedback page.
