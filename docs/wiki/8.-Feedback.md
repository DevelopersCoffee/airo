# Feedback

Airo is actively evolving. Use GitHub Issues for feedback:

- [Report a bug](https://github.com/DevelopersCoffee/airo/issues/new?labels=bug)
- [Suggest a feature](https://github.com/DevelopersCoffee/airo/issues/new?labels=enhancement)
- [View existing issues](https://github.com/DevelopersCoffee/airo/issues)

When reporting a bug, include:

- Platform: Android, iOS, Web, Windows, Linux, or macOS.
- App version or commit.
- Steps to reproduce.
- Expected behavior.
- Actual behavior.
- Screenshots or logs if available.
