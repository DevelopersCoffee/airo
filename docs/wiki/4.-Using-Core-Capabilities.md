# Using Core Capabilities

## AI Chat

Use the Assistant tab for open-ended chat and supported app commands.

1. Open **Assistant**.
2. Type a message or tap a prompt card.
3. Airo first checks whether the message matches a supported app action.
4. If no local command matches, Airo asks the AI service for a response.

Examples:

- `Help me think through a task`
- `Make me a 7 day vegetarian diet plan`
- `Create a morning study routine for tomorrow`

## Split Bill

Use chat for quick equal splits or open the full bill split flow.

Examples:

- `Split this ₹2400 bill with Asha, Ben and Chen`
- Open **Dashboard** then **Split Bill** for receipt or participant details.

## Money Dashboard

Use Dashboard for finance workflows:

- View total balance.
- Review recent transactions.
- Track budgets.
- Add expenses.
- Manage groups.
- Add shared splits inside a group.

## Quest File Analysis

Use Tasks when you want to work with files.

1. Open **Tasks**.
2. Tap **Create Quest** or the add button.
3. Enter a title.
4. Upload one or more supported files: PDF, JPG, JPEG, PNG, TXT, DOC, or DOCX.
5. Continue into the quest chat.

## Ask Image

The Assistant routes image-analysis intent to the Quest upload flow.

1. Open **Assistant**.
2. Use the **Ask Image** card or ask about an image/receipt.
3. Airo opens **Tasks > New Quest** so the image can be uploaded and discussed.

## Audio Scribe

Audio Scribe is planned as an offline transcription workflow. The current
Assistant response maps this to Airo voice workflows and the future
meeting-minutes pipeline.

Saved meeting intelligence search is validated locally against representative
edge cases. The current in-memory contract supports title and project lookup,
speaker-label lookup, representative typo recovery, mixed-language tokens, and
stable deterministic ordering for repeated queries.

Saved meeting intelligence also generates structured local sections for summary,
decisions, risks, open questions, follow-ups, blockers, dependencies, and next
steps from final redacted transcript content.

## Mobile Actions

The Assistant can explain or route supported Airo actions. Sensitive device
actions should remain confirmation-gated before execution.

## Model Management

Use the **Model Management** prompt card or Assistant profile to open AI model
settings.

## Arena Games

Use **Games** or the Assistant.

Examples:

- `I am bored, start chess`
- `Play chess`
- Open **Games** and choose an available game.

Available games documented today:

- Chess
- Blackjack

Planned or documented upcoming games include Texas Hold'em, Poker, Rummy, and
Solitaire.
