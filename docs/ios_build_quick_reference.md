## Quick Start

### First Time Setup

```bash
# 1. Verify SPM is enabled (should already be done)
flutter config --list | grep swift-package-manager
# Should show: enable-swift-package-manager: true

# 2. Get dependencies
cd app
flutter pub get

# 3. Build
flutter build ios --simulator --debug --no-codesign
```

### Daily Development

```bash
# Clean build
flutter clean && flutter pub get

# Run on simulator
flutter run

# Run on device
flutter run -d <device-id>

# Build release
flutter build ios --release
```

## Troubleshooting

### "CocoaPods not installed" Warning

**This is normal!** Flutter still shows this warning even with SPM enabled. Ignore it.

### Build Fails

```bash
# Nuclear option - clean everything
flutter clean
rm -rf app/ios/Flutter/ephemeral/
flutter pub get
flutter build ios --simulator --debug --no-codesign
```

### Plugin Not Found

```bash
# Verify SPM package exists
ls app/ios/Flutter/ephemeral/Packages/FlutterGeneratedPluginSwiftPackage/

# If missing, regenerate
flutter pub get
```
