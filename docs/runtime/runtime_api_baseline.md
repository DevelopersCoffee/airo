# Runtime API Baseline

With the conclusion of Program 1 and certification via PFR-2, the following API surfaces are strictly **FROZEN**. Any further modification requires an accepted Architecture Decision Record (ADR).

### Frozen Packages
- `platform_engine_sdk`: The contract boundaries for all models.
- `platform_delegates`: Hardware and capability resolutions.
- `platform_native`: Native FFI lifecycle abstractions.
- `platform_runtime`: Runtime coordination, model routing, and dependency abstractions.
