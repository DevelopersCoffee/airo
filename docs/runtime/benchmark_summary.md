# Benchmark Summary

This summary serves as the reporting schema. Actual performance numbers are dynamic and depend entirely on the target host hardware.

### Tracked Metrics
- **Load Time**: Time from instantiation to first ready state.
- **TTFT (Time To First Token)**: Latency of streaming generation.
- **Tokens/Sec**: Sustained throughput latency.
- **Peak Memory**: Resident Set Size (RSS) during peak inference load.

*Note: Benchmarks are informative, not normative. Compliance does not dictate performance equivalence.*
